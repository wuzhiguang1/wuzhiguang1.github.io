еҪ“жҲ‘д»¬иҫ“е…Ҙ npm install зҡ„ж—¶еҖҷпјҢиҷҪз„¶жІЎжңүеңЁеҗҺйқўеҠ жЁЎеқ—еҗҚпјҢе®ғдјҡиҮӘеҠЁеҺ» package.json дёӯеҺ»еҜ»жүҫпјҢжҲ‘д»¬д№ӢеүҚе®үиЈ…иҝҮзҡ„ дҫқиө–йЎ№пјҢз„¶еҗҺжҠҠиҝҷдәӣдҫқиө–йЎ№е®үиЈ…гҖӮ


2 гҖҒиҝһзӯүиөӢеҖј

иҫ“еҮәдёҖдёӢд»Јз Ғзҡ„жү§иЎҢз»“жһңпјҢе№¶и§ЈйҮҠдёәд»Җд№Ҳ

иҝҷйҒ“йўҳиҖғзҡ„е°ұжҳҜжҲ‘д»¬еҜ№иҝһзӯүиөӢеҖјзҡ„зҗҶи§ЈгҖӮ

иҝһзӯүејҖе§Ӣд№ӢеүҚпјҢзЁӢеәҸдјҡжҠҠжүҖжңүеј•з”ЁйғҪдҝқеӯҳдёӢжқҘпјҢиҝһзӯүзҡ„иҝҮзЁӢдёӯпјҢиҝҷдәӣеҖјжҳҜдёҚеҸҳзҡ„пјҢзӯүеҲ°иҝһзӯүйғҪе®ҢдәӢдәҶпјҢеҶҚдёҖеқ—еҸҳгҖӮ

жҲ‘д»¬жқҘзңӢдёҖдёӢиҝҷдёӘеӣҫпјҡ

в‘  a жҢҮеҗ‘ дәҶ{n:1}

в‘Ў b зӯүдәҺ a пјҢеҲҷbзҡ„еҶ…еӯҳд№ҹжҢҮеҗ‘дәҶ {n:1}

в‘ў иҝҷжҳҜиҝһзӯүиөӢеҖјпјҢиҝһзӯүејҖе§Ӣд№ӢеүҚпјҢзЁӢеәҸдјҡжҠҠжүҖжңүеј•з”ЁйғҪдҝқеӯҳдёӢжқҘпјҢиҝһзӯүзҡ„иҝҮзЁӢдёӯпјҢиҝҷдәӣеҖјжҳҜдёҚеҸҳзҡ„пјҢзӯүеҲ°иҝһзӯүйғҪе®ҢдәӢдәҶпјҢеҶҚдёҖеқ—еҸҳпјҢзЁӢеәҸд»ҺеҸіеҫҖе·ҰпјҢе…ҲзңӢa = {n:2} д№ҹе°ұжҳҜеҰӮеӣҫзҡ„в‘ў.1 пјҢa жҢҮеҗ‘дәҶ {n:2}пјҢ

дҪҶжҳҜз”ұдәҺжҳҜиҝһзӯүиөӢеҖјпјҢжӯӨж—¶ a иҝҳжҳҜдҝқжҢҒиҝҷеҜ№ {n:1} зҡ„еј•з”ЁгҖӮз„¶еҗҺеҶҚзңӢ a.x = a д№ҹе°ұжҳҜеҰӮеӣҫзҡ„ в‘ў.2 , еңЁaеј•з”Ёзҡ„еҶ…еӯҳең°еқҖдёӯ ж·»еҠ дёҖдёӘ x еұһжҖ§пјҢе№¶иөӢеҖјдёә {n:2}гҖӮеҶҚз„¶еҗҺ еҰӮеӣҫв‘ў.3 зӯүеҲ°ж•ҙдёӘиҜӯеҸҘе®ҢжҲҗеҗҺ,aдёҚеҶҚжҢҮеҗ‘{n:1}пјҢиҖҢжҢҮеҗ‘дәҶ {n:2}гҖӮ

a=b=c=value

3гҖҒж•°з»„йҮҢйқўжңү10дёҮдёӘж•°жҚ®пјҢеҸ–з¬¬дёҖдёӘе…ғзҙ е’Ңз¬¬10дёҮдёӘе…ғзҙ зҡ„дәӢд»¶зӣёе·®еӨҡе°‘пјҹ

еӣһзӯ”пјҡ е®Ңе…ЁдёҖж ·
