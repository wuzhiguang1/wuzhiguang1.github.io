1.2 жҺЁйҖҒ

ејҖеҸ‘дҝ®ж”№е®ҢжҠҠжң¬ең°еә“зҡ„ж–Үд»¶жҺЁйҖҒеҲ°иҝңзЁӢд»“еә“ еүҚжҸҗжҳҜжҸҗдәӨеҲ°дәҶжң¬ең°еә“жүҚеҸҜд»ҘжҺЁйҖҒ

1.3 е…ӢйҡҶ

е®Ңж•ҙзҡ„жҠҠиҝңзЁӢеә“е…ӢйҡҶеҲ°жң¬ең° е…ӢйҡҶдёӢжқҘеҗҺдёҚиҰҒеңЁдё»еҲҶж”ҜйҮҢйқўеҒҡејҖеҸ‘ cloneиҝӣиЎҢдёҖж¬ЎпјҢд»Һж— еҲ°жңүзҡ„иҝҮзЁӢпјҢжӣҙж–°з”Ёpull

1.4 жӢүеҸ–

жң¬ең°еӯҳеңЁcloneдёӢжқҘзҡ„ж–Үд»¶ е°ұз”Ёpullжӣҙж–°

жіЁж„Ҹпјҡ еҰӮжһңдҪ жҸҗдәӨжҲҗеҠҹдёӢзҸӯеҗҺпјҡеҗҢдәӢд№ҹеҗ‘иҜҘиҝңзЁӢд»“еә“жҸҗдәӨдәҶж–°зҡ„еҶ…е®№гҖӮз¬¬дәҢеӨ©дёҠзҸӯж—¶пјҢдҪ йңҖиҰҒжӢүеҺ»зәҝдёҠзҡ„д»“еә“зҡ„жңҖж–°зүҲжң¬гҖӮ

еӣ жӯӨеңЁжҜҸеӨ©е·ҘдҪңзҡ„з¬¬дёҖд»¶дәӢе°ұжҳҜgit pull жӢүеҸ–зәҝдёҠзҡ„жңҖж–°зүҲжң¬пјҢдёӢзҸӯиҰҒеҒҡзҡ„е°ұжҳҜ git pushпјҢе°Ҷжң¬ең°д»Јз ҒжҸҗдәӨеҲ°зәҝдёҠд»“еә“гҖӮ

1.5 и§ЈеҶіеҶІзӘҒ

жіЁж„Ҹпјҡи§ЈеҶіеҶІзӘҒеҗҺзҡ„жҸҗдәӨжҳҜдёҚиғҪеёҰж–Үд»¶еҗҚзҡ„

Gitеҹәжң¬зҗҶи®әпјҲйҮҚиҰҒпјү

1.1 дёүдёӘеҢәеҹҹ

Gitжң¬ең°жңүдёүдёӘе·ҘдҪңеҢәеҹҹпјҡе·ҘдҪңзӣ®еҪ•пјҲWorking DirectoryпјүгҖҒжҡӮеӯҳеҢә(Stage/Index)гҖҒиө„жәҗеә“(RepositoryжҲ–Git Directory)гҖӮеҰӮжһңеңЁеҠ дёҠиҝңзЁӢзҡ„gitд»“еә“(Remote Directory)е°ұеҸҜд»ҘеҲҶдёәеӣӣдёӘе·ҘдҪңеҢәеҹҹгҖӮж–Үд»¶еңЁиҝҷеӣӣдёӘеҢәеҹҹд№Ӣй—ҙзҡ„иҪ¬жҚўе…ізі»еҰӮдёӢпјҡ

  • Workspaceпјҡе·ҘдҪңеҢәпјҢе°ұжҳҜдҪ е№іж—¶еӯҳж”ҫйЎ№зӣ®д»Јз Ғзҡ„ең°ж–№
  • Index / StageпјҡжҡӮеӯҳеҢәпјҢз”ЁдәҺдёҙж—¶еӯҳж”ҫдҪ зҡ„ж”№еҠЁпјҢдәӢе®һдёҠе®ғеҸӘжҳҜдёҖдёӘж–Үд»¶пјҢдҝқеӯҳеҚіе°ҶжҸҗдәӨеҲ°ж–Үд»¶еҲ—иЎЁдҝЎжҒҜ
  • Repositoryпјҡд»“еә“еҢәпјҲжҲ–жң¬ең°д»“еә“пјүпјҢе°ұжҳҜе®үе…Ёеӯҳж”ҫж•°жҚ®зҡ„дҪҚзҪ®пјҢиҝҷйҮҢйқўжңүдҪ жҸҗдәӨеҲ°жүҖжңүзүҲжң¬зҡ„ж•°жҚ®гҖӮе…¶дёӯHEADжҢҮеҗ‘жңҖж–°ж”ҫе…Ҙд»“еә“зҡ„зүҲжң¬
  • RemoteпјҡиҝңзЁӢд»“еә“пјҢжүҳз®Ўд»Јз Ғзҡ„жңҚеҠЎеҷЁпјҢеҸҜд»Ҙз®ҖеҚ•зҡ„и®ӨдёәжҳҜдҪ йЎ№зӣ®з»„дёӯзҡ„дёҖеҸ°з”өи„‘з”ЁдәҺиҝңзЁӢж•°жҚ®дәӨжҚў

жң¬ең°зҡ„дёүдёӘеҢәеҹҹзЎ®еҲҮзҡ„иҜҙеә”иҜҘжҳҜgitд»“еә“дёӯHEADжҢҮеҗ‘зҡ„зүҲжң¬пјҡ

  • DirectoryпјҡдҪҝз”ЁGitз®ЎзҗҶзҡ„дёҖдёӘзӣ®еҪ•пјҢд№ҹе°ұжҳҜдёҖдёӘд»“еә“пјҢеҢ…еҗ«жҲ‘д»¬зҡ„е·ҘдҪңз©әй—ҙе’ҢGitзҡ„з®ЎзҗҶз©әй—ҙгҖӮ
  • WorkSpaceпјҡйңҖиҰҒйҖҡиҝҮGitиҝӣиЎҢзүҲжң¬жҺ§еҲ¶зҡ„зӣ®еҪ•е’Ңж–Үд»¶пјҢиҝҷдәӣзӣ®еҪ•е’Ңж–Үд»¶з»„жҲҗдәҶе·ҘдҪңз©әй—ҙгҖӮ
  • .gitпјҡеӯҳж”ҫGitз®ЎзҗҶдҝЎжҒҜзҡ„зӣ®еҪ•пјҢеҲқе§ӢеҢ–д»“еә“зҡ„ж—¶еҖҷиҮӘеҠЁеҲӣе»әгҖӮ
  • Index/StageпјҡжҡӮеӯҳеҢәпјҢжҲ–иҖ…еҸ«еҫ…жҸҗдәӨжӣҙж–°еҢәпјҢеңЁжҸҗдәӨиҝӣе…Ҙrepoд№ӢеүҚпјҢжҲ‘д»¬еҸҜд»ҘжҠҠжүҖжңүзҡ„жӣҙж–°ж”ҫеңЁжҡӮеӯҳеҢәгҖӮ
  • Local Repoпјҡжң¬ең°д»“еә“пјҢдёҖдёӘеӯҳж”ҫеңЁжң¬ең°зҡ„зүҲжң¬еә“пјӣHEADдјҡеҸӘжҳҜеҪ“еүҚзҡ„ејҖеҸ‘еҲҶж”ҜпјҲbranchпјүгҖӮ
  • Stashпјҡйҡҗи—ҸпјҢжҳҜдёҖдёӘе·ҘдҪңзҠ¶жҖҒдҝқеӯҳж ҲпјҢз”ЁдәҺдҝқеӯҳ/жҒўеӨҚWorkSpaceдёӯзҡ„дёҙж—¶зҠ¶жҖҒгҖӮ

пј“гҖҒе°ҶжҡӮеӯҳеҢәеҹҹзҡ„ж–Үд»¶жҸҗдәӨеҲ°gitд»“еә“гҖӮ

еӣ жӯӨпјҢgitз®ЎзҗҶзҡ„ж–Үд»¶жңүдёүз§ҚзҠ¶жҖҒпјҡе·Ідҝ®ж”№пјҲmodifiedпјү,е·ІжҡӮеӯҳпјҲstagedпјү,е·ІжҸҗдәӨ(committed)

1.3 ж–Үд»¶зҡ„еӣӣз§ҚзҠ¶жҖҒ

зүҲжң¬жҺ§еҲ¶е°ұжҳҜеҜ№ж–Үд»¶зҡ„зүҲжң¬жҺ§еҲ¶пјҢиҰҒеҜ№ж–Үд»¶иҝӣиЎҢдҝ®ж”№гҖҒжҸҗдәӨзӯүж“ҚдҪңпјҢйҰ–е…ҲиҰҒзҹҘйҒ“ж–Үд»¶еҪ“еүҚеңЁд»Җд№ҲзҠ¶жҖҒпјҢдёҚз„¶еҸҜиғҪдјҡжҸҗдәӨдәҶзҺ°еңЁиҝҳдёҚжғіжҸҗдәӨзҡ„ж–Үд»¶пјҢжҲ–иҖ…иҰҒжҸҗдәӨзҡ„ж–Үд»¶жІЎжҸҗдәӨдёҠгҖӮ

  • Untracked: жңӘи·ҹиёӘ, жӯӨж–Үд»¶еңЁж–Үд»¶еӨ№дёӯ, дҪҶе№¶жІЎжңүеҠ е…ҘеҲ°gitеә“, дёҚеҸӮдёҺзүҲжң¬жҺ§еҲ¶. йҖҡиҝҮgit add зҠ¶жҖҒеҸҳдёәStaged.
  • Unmodify: ж–Үд»¶е·Із»Ҹе…Ҙеә“, жңӘдҝ®ж”№, еҚізүҲжң¬еә“дёӯзҡ„ж–Үд»¶еҝ«з…§еҶ…е®№дёҺж–Үд»¶еӨ№дёӯе®Ңе…ЁдёҖиҮҙ. иҝҷз§Қзұ»еһӢзҡ„ж–Үд»¶жңүдёӨз§ҚеҺ»еӨ„, еҰӮжһңе®ғиў«дҝ®ж”№, иҖҢеҸҳдёәModified. еҰӮжһңдҪҝз”Ёgit rmз§»еҮәзүҲжң¬еә“, еҲҷжҲҗдёәUntrackedж–Үд»¶
  • Modified: ж–Үд»¶е·Ідҝ®ж”№, д»…д»…жҳҜдҝ®ж”№, е№¶жІЎжңүиҝӣиЎҢе…¶д»–зҡ„ж“ҚдҪң. иҝҷдёӘж–Үд»¶д№ҹжңүдёӨдёӘеҺ»еӨ„, йҖҡиҝҮgit addеҸҜиҝӣе…ҘжҡӮеӯҳstagedзҠ¶жҖҒ, дҪҝз”Ёgit checkout еҲҷдёўејғдҝ®ж”№иҝҮ, иҝ”еӣһеҲ°unmodifyзҠ¶жҖҒ, иҝҷдёӘgit checkoutеҚід»Һеә“дёӯеҸ–еҮәж–Үд»¶, иҰҶзӣ–еҪ“еүҚдҝ®ж”№ !
  • Staged: жҡӮеӯҳзҠ¶жҖҒ. жү§иЎҢgit commitеҲҷе°Ҷдҝ®ж”№еҗҢжӯҘеҲ°еә“дёӯ, иҝҷж—¶еә“дёӯзҡ„ж–Үд»¶е’Ңжң¬ең°ж–Үд»¶еҸҲеҸҳдёәдёҖиҮҙ, ж–Үд»¶дёәUnmodifyзҠ¶жҖҒ. жү§иЎҢgit reset HEAD filenameеҸ–ж¶ҲжҡӮеӯҳ, ж–Үд»¶зҠ¶жҖҒдёәModified
