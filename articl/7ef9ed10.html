27гҖҒ AMDпјҲModules/Asynchronous-DefinitionпјүгҖҒCMDпјҲCommon Module Definitionпјүи§„иҢғеҢәеҲ«пјҹ

AMD жҳҜ RequireJS еңЁжҺЁе№ҝиҝҮзЁӢдёӯеҜ№жЁЎеқ—е®ҡд№үзҡ„и§„иҢғеҢ–дә§еҮәгҖӮCMD жҳҜ SeaJS еңЁжҺЁе№ҝиҝҮзЁӢдёӯеҜ№жЁЎеқ—е®ҡд№үзҡ„и§„иҢғеҢ–дә§еҮәгҖӮ
еҢәеҲ«пјҡ

  1. еҜ№дәҺдҫқиө–зҡ„жЁЎеқ—пјҢAMD жҳҜжҸҗеүҚжү§иЎҢпјҢCMD жҳҜе»¶иҝҹжү§иЎҢгҖӮдёҚиҝҮ RequireJS д»Һ 2.0 ејҖе§ӢпјҢд№ҹж”№жҲҗеҸҜд»Ҙе»¶иҝҹжү§иЎҢпјҲж №жҚ®еҶҷжі•дёҚеҗҢпјҢеӨ„зҗҶж–№ејҸдёҚеҗҢпјүгҖӮ
  2. CMD жҺЁеҙҮдҫқиө–е°ұиҝ‘пјҢAMD жҺЁеҙҮдҫқиө–еүҚзҪ®гҖӮ
  3. AMD зҡ„ API й»ҳи®ӨжҳҜдёҖдёӘеҪ“еӨҡдёӘз”ЁпјҢCMD зҡ„ API дёҘж јеҢәеҲҶпјҢжҺЁеҙҮиҒҢиҙЈеҚ•дёҖгҖӮ
